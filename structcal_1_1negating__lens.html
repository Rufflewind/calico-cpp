Detailed Description

template<class Lens, class T = typename lens_traits<Lens>::value_type>
struct cal::negating_lens< Lens, T >

A lens that negates the value before storing or accessing it.

If L is the negating_lens and x is the internal lens, the following relationships are satisfied: 

L        ==   -x
L  = c  <==>   x  = -c
L += c  <==>   x -=  c
L -= c  <==>   x +=  c

The addition/subtraction-assignment operators are overloaded here to ensure better optimization for floating-point numbers (when -ffast-math or its equivalent is not enabled).
